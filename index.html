Selection for Top Designs recognises exceptional technical complexity, design quality, and documentation, and represents one of the highest distinctions achievable in a VCE technology subject.

Multiprocessor Embedded Vision System on FPGA (2024)

I developed a high performance embedded vision system on the DE10 Lite FPGA, transitioning a single processor design into a fully parallel multiprocessor architecture. The systemcaptured image data from an ESP32 CAM over SPI, applied real time filters, and output live video over VGA. Key Contributions:

Designed and implemented the shared SDRAM memory architecture used for interprocessor communication, including synchronisation logic using mutexes, message passing, and safe read write patterns.

Built the SPI data acquisition pipeline, enabling continuous capture of packed and RGB frame data, and integrating sensor inputs for real time orientation control.

Implemented custom hardware accelerated image processing instructions in Verilog, including grayscale conversion, Gaussian blur, and Sobel edge detection, reducing per pixel compute time to single cycle operations.

Helped re architect the system across three processors (Master, SPI, Display), enabling true concurrency between frame capture, processing, and rendering.

Delivered measurable system gains, increasing frame rate from the 1.5 to 2 FPS baseline to more than 3 FPS in single image mode and over 7 FPS in quad mode, while supporting new RGB functionality and advanced filters.
